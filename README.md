# NVX-Bridge-Release

Этот репозиторий отвечает за публикацию официальных сборок **NVX-Bridge**. Исходного кода здесь нет: в дереве лежат только лицензии, NOTICE и workflow GitHub Actions, который подтягивает приватный репозиторий `NVX-Bridge`, собирает бинарии и оформляет релизы.

## Что попадает в релизные архивы
- исполнимый файл `NVX-Bridge-<platform>` (или `.exe` для Windows), собранный из `cmd/nvx-bridge`;
- `LICENSE`, `NOTICE` и каталог `licenses/` с текстами зависимостей;
- `REVISION.txt` с укороченным SHA коммита, из которого собрана версия.
