# NVX-Bridge

Мост между сетью устройств и настольной лабораторией: одно приложение объединяет мульти-панельный GUI, диспетчер устройств. Причем, NVX-Bridge создает единое адресное пространство всех устройств, наподобие VISA (National Instruments), которые далее могут управляться по SCPI протоколу. Среди стандартных VISA устройств, особое место занимают устройства на базе прошивки NanoVNA-X (USB VID `0x0483`, PID `0x5740`). GUI NVX-Bridge использует NanoVNA-X протокол, предоставляет элементы управления, конфигурации, управление свипами/калибровкой, чтобы работать параллельно с множественными устройствами подобного рода, предоставляя им пользовательский интерфейс.
Далее, как и обычные VISA устройства, они представляются через SCPI-сервер на TCP :5025,
